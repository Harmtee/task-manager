## About Laravel

This is a task manager built with Laravel and jQuery. It include ability to prioritize task item by sorting them through dragging of item in the user interface  


## Installation
Open/create the .env file in the project and edit the database variables as desired. To install this project run the following in your terminal.

composer install
composer dump-autoload
php artisan migrate
npm install
nmp run dev

Start the application by running
php artisan serve
